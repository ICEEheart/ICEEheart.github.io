博客搭建

####博客搭建
1.下载node.js 网址:nodejs.org

2.安装打开cmd查看node -v    
               npm -v

3.安装淘宝镜像源(管理员权限运行):npm install -g cnpm --registry=https:registry.npm.taobao.org

4.安装成功后利用cnpm安装hexo博客框架cnpm install -g hexo-cli

5.创建一个空文件夹(一定的是空的)然后cd 到该目录下输入hexo init指令即可。成功之后可看到该文件夹下多了很多文件。

6.启动hexo  运行指令hexo s    s(start简写)

7.打开浏览器输入http://localhost:4000 看到博客页面

8.创建一篇文章可以用hexo n "我的第一篇博客文章"指令

9.找到你创建文件夹下的\source\_posts  你会发现该文件

10.打开进行编辑

11.编辑好后输入指令hexo g进行提交，提交之后在执行第步就可看到你写的文章

博客部署到github上

1.到自己的github上创建一个新的仓库 遵循命名规则github用户名.github.io  后点击create即可

2.安装git插件到博客目录下  执行命令cnpm install --save hexo-deployer-git 

3.设置_config.yml文件   找到#Deployment  在type: git
                                    新添加repo:将刚才创建的仓库地址放此处即可
                                    例如:repo: https://github.com/ICEEheart/ICEEheart.github.io.git
                                    新添加branch: master保存退出
4.将hexo部署到远端 执行命令:hexo d即可

更换主题

1.访问地址:https://github.com/litten/hexo-theme-yilia

2.git clone 到theme/yilia目录下

3.更改_config.yml文件中theme:yilia

4.远程提交:hexo d
